# HR Analytics Dashboard 📊
## Project Overview
This HR Analytics Dashboard project helps analyze employee attrition, workforce demographics, and department-wise insights using interactive visualizations. The dashboard provides meaningful business insights that help understand employee trends and attrition patterns.
---
## Key Insights
Total Employees: 1.47K
 * Active Employees: 1.238K
 * Attrition Count: 236
 * Attrition Rate: 16.0%
 * Average Age: 36.97
 * Avg Experience: 7.02
 * Attrition by Department
 * Attrition by Salary Slab
 * Attrition by Job Role & Job Satisfaction
 * Age Group Distribution
 * Attrition by Gender
 * Attrition Trend by Experience
 * Department-wise Employee Count
